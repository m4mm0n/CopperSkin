# Baseline governance

The repository baseline for this work is the source tree, `CopperSkin.slnx`, `Directory.Build.props`, existing test projects, `README.md`, `RELEASE_NOTES.md`, and `docs/THEME_FORMAT.md`. The v0.3 graphics ownership boundary is recorded in [ADR-0001](adr/ADR-0001-v0.3-graphics-boundary.md). Changes that affect public packaging, compatibility, signing, or release automation must update the deployment plan and the release notes together.
